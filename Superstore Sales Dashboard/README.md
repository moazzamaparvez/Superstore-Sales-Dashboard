# Superstore Data Analytics Portfolio

## Overview
This portfolio demonstrates data analytics and visualization skills using a **Superstore dataset**. The goal is to showcase insights derived from sales data across regions, categories, time periods, and customers using **Excel, PivotTables, charts, and dashboards**.


## Tools Used
- **Excel:** PivotTables, Charts, Slicers  
- **Power Query:** Used for loading and cleaning data (TRIM, CLEAN, removing duplicates, etc.)  


## Dataset
- Original dataset: **Kaggle Superstore Dataset (CSV format)**  
- Cleaned and saved as: `sales_data_clean.xlsx`  

## Data Cleaning & Preparation

### Text Columns Cleaned
- Customer Name, Product Name, Category, Customer ID, ZIP Code → trimmed, standardized, duplicates removed  

### Numeric Columns Cleaned
- Sales → formatted to 2 decimal places  
- Profit → formatted to 2 decimal places (negative values kept)  
- Quantity → formatted as whole numbers  
- Discount → formatted to 2 decimal places  

### Other Steps
- Checked for missing values and inconsistencies  
- Removed duplicate records  
- Standardized text using **TRIM** and **Find & Replace**
